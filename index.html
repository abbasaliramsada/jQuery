1. Introduction to jQuery

jQuery is a fast, small, and feature-rich JavaScript library.

How to Include jQuery

<!-- Using CDN -->
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

<!-- Or download and include locally -->
<script src="jquery-3.6.0.min.js"></script>

Document Ready

Ensures code runs after DOM is fully loaded

// Shorthand version (most common)
$(function() {
    // Your code here
});

// Long version
$(document).ready(function() {
    // Your code here
});

2. Basic jQuery Syntax

The basic syntax is: $(selector).action()

Simple Example

This is a paragraph that can be hidden/shown.

// Hide paragraph when button is clicked
$("#hide-btn").click(function() {
    $("#demo-para").hide(); // Hides the element
});

// Show paragraph when button is clicked
$("#show-btn").click(function() {
    $("#demo-para").show(); // Shows the element
});

3. jQuery Selectors

jQuery uses CSS-style selectors to select HTML elements.

Basic Selectors

Paragraph with class "test"

Another paragraph without class

  • List item 1
  • List item 2
// Element selector
$("p").css("color", "blue"); // All paragraphs

// ID selector
$("#demo-div").css("border", "2px solid red");

// Class selector
$(".test").css("font-weight", "bold");

// Multiple elements
$("p, li").css("font-size", "18px");

// Child selector
$("ul li").css("color", "green");

4. jQuery Events

jQuery provides simple methods for handling events.

Common Events

Hover over me 
// Click event
$("#click-btn").click(function() {
    alert("Button was clicked!");
});

// Hover event (mouseenter + mouseleave)
$("#hover-div").hover(
    function() {
        $(this).css("background", "#ddd");
    },
    function() {
        $(this).css("background", "#eee");
    }
);

// Keypress event
$("#key-input").keypress(function() {
    console.log("You pressed a key");
});

5. jQuery Effects

jQuery provides several effect methods.

Basic Effects

// Fade effects
$("#fade-btn").click(function() {
    $("#effects-div").fadeToggle(1000); // 1000ms = 1s
});

// Slide effects
$("#slide-btn").click(function() {
    $("#effects-div").slideToggle();
});

// Custom animation
$("#animate-btn").click(function() {
    $("#effects-div").animate({
        width: "toggle",
        opacity: 0.5,
        height: "150px"
    }, 1500);
});

6. DOM Manipulation

jQuery makes it easy to manipulate the DOM.

Content Manipulation

Original content 
// Change HTML content
$("#html-btn").click(function() {
    $("#dom-div").html("<b>New HTML content</b>");
});

// Change text content
$("#text-btn").click(function() {
    $("#dom-div").text("New text content");
});

// Get input value
$("#val-btn").click(function() {
    alert($("#name-input").val());
});

// Append content
$("#append-btn").click(function() {
    $("#dom-div").append(" <span style='color:red;'>(appended)</span>");
});

CSS and Attributes

jQuery Website 
// Change CSS
$("#css-btn").click(function() {
    $("#css-div").css({
        "background": "#2ecc71",
        "border-radius": "50%"
    });
});

// Get attribute
$("#attr-btn").click(function() {
    alert($("#jquery-link").attr("href"));
});

7. jQuery AJAX

jQuery simplifies AJAX requests.

Basic AJAX Methods

AJAX response will appear here
// Load content from a file
$("#load-btn").click(function() {
    $("#ajax-content").load("demo_test.html");
});

// GET request
$("#get-btn").click(function() {
    $.get("demo_test.asp", function(data, status) {
        $("#ajax-content").html("Data: " + data + "\nStatus: " + status);
    });
});

// POST request
$("#post-btn").click(function() {
    $.post("demo_test_post.asp",
    {
        name: "Donald Duck",
        city: "Duckburg"
    },
    function(data, status) {
        $("#ajax-content").html("Data: " + data + "\nStatus: " + status);
    });
});

8. Advanced jQuery Concepts

Method Chaining

// Multiple methods in one statement
$("#chain-btn").click(function() {
    $("#chain-div")
        .css("background", "#16a085")
        .slideUp(1000)
        .slideDown(1000)
        .animate({width: "200px"});
});

Event Delegation

  • Existing item
// Works for current AND future elements
$("#event-ul").on("click", "li", function() {
    $(this).toggleClass("highlight");
});

$("#add-li-btn").click(function() {
    $("#event-ul").append("<li>New item (click me)</li>");
});

Promises and Deferreds

// Execute code after animations complete
$("#promise-btn").click(function() {
    $(".box")
        .first()
        .fadeOut(1000)
        .fadeIn(1000)
        .promise()
        .done(function() {
            $(".box")
                .eq(1)
                .animate({left: "+=100"}, 1000)
                .promise()
                .done(function() {
                    $(".box")
                        .last()
                        .animate({height: "toggle"}, 1000);
                });
        });
});
